# How to Start the Application
### Step1: Clone the Repo From Github
### Step2: Install Application Dependencies

**For Windows:**
- Install Node.js 
- Install npm

After installation, you can check npm and Node.js versions by using the commands below in the command prompt:
`node -v`
`npm -v`

**For MAC:**
- Install Homebrew
- install Node.js 


After installation, you can check npm and Node.js versions by using the commands below in the command prompt:
`node -v`
`npm -v`

### Step 3: Run the Application
Open Command Prompt and run `npm start`
- cd to "client" directory
- npm start 
 
### Application on Web
Once we did all the steps, our application will start running. <br />
Application should be running on **localhost::3000**

Heroku Deployment: https://easy-estate-80134be2da71.herokuapp.com/

# Users can
- Search for their dream house.
- Sign up for an account.
- Log in to their existing account.
- Log in/sign up with email or with Google.
- Select any location in NY.
- Choose house categories:
    - Multi Family
    - Single Family
    - Mobile
    - Land
    - Farm 
- Choose their minimum and maximum price.
